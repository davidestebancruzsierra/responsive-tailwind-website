# responsive-tailwind-website

## Overview
This project consist of a **web page** about **Belajar Responsive Website**.

## Objectives
Learn and practice developing a static website using the CSS Tailwind framework, ensuring that the website is responsive for mobile, tablet, and desktop, and has animations and transitions.

### **Frontend Languages**
- **HTML5** — semantic structure for all pages.
- **CSS3 (Tailwind CSS utilities)** — styling using utility classes only.
- **Minimal Custom CSS** — only for animations and variables (as allowed).

### **Tools & Build System**
- **Node.js** — environment required for Tailwind.
- **npm** — package management.
- **Tailwind CSS v3** — main styling library.
- **PostCSS** — processing Tailwind output.
- **Git & GitHub** — version control and repository hosting.

### **Terminal Tools**
- **Git Bash** — to run Tailwind in watch mode and manage build commands.
